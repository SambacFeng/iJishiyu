# iJishiyu

西南大学计算机与信息科学学院团委下属及时雨志愿者服务队，i及时雨微信小程序，面向西南大学全体师生提供笔记本报修表单收集功能，同时具有面向内部进行的部门工作管理、数据统计等功能

使用微信小程序原生语法+第三方组件库 [Lin-UI](https://github.com/TaleLin/lin-ui)
